Intro To JavaScript

Variables & Simple Data

In JavaScript we declare a variable using the keyword "var". This is used for all data types. A single "=" then assigns the variable some value. To print something we can use the "console.log" statement.

A method in JavaScript are actions that can be performed on objects. Methods are functions stored as object properties. The first method we are introduced to is "string.replace()". This method finds a pattern in a string and replaces some or all matches of it. To read the documentation on it, click here. We can also use JavaScript methods to replace place holders in HTML with real data, then we can append HTML strings. In the videos they used the ".append" and ".slice" method to do so, additional documentation on append and on slice.

In JavaScript we work alot with true or false.However, if something is neither true or false it's "undefined". We also will get to know how to use arrays. They are similar to lists in Python. An array is a special kind of object.

So then, what are objects? They hold information and do things. They can be declared in a few different ways. Unlike Python, there are no classes in JavaScript when working with objects. Object literal notation uses curly braces with key value pairs inside. Dot notation allows us to declare and object and then define its properties outside of the curly braces. There is no need to use var because we are adding properties to an object, not creating new variables. Bracket notation is similar to dot notation in that we can declare and object and then define its properties outside of the braces.
